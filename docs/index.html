CSScaffold Documentation

Overview

Everyday we work with CSS and try to workaround its shortcomings. We've tried building frameworks, and more frameworks, and even more frameworks. But this 'framework' word gets thrown around so much these day you can call just about anything a framework.

I think the word people were really looking for when they made these 'frameworks' was 'template' or just 'really big reset you'll override later'. What a real framework should do is make common tasks easier by abstracting the difficulty out of them and put the bulk of the work onto the unseen back-end.

Basic Usage

Grid

The math plugin allows you to perform some simple math equations inside your css. This is particularly useful for calculating the widths of baselines and other grid specific widths.

You can use the grid variables inside the math function, so you can calculate accurate widths.

#id
{
	width: math('grid(baseline) * 2')px;
	padding: math('10 * 2')px;
}
