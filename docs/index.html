CSScaffold Documentation and Tests

These pages are tied directly to your work CSS, so when you've styled up these, your site will have a head start!

These pages are used to build the foundation of your css before you begin mirroring your design. The goal of these pages is to get you to have you layouts, content, type and forms ready before you begin creating your site.

For example, if you get every element styled on the typography page according to the design, when you create your pages, you shouldn't have to touch your type styles. This results in a much more consistent site

Math

The math plugin allows you to perform some simple math equations inside your css. This is particularly useful for calculating the widths of baselines and other grid specific widths.

You can use the grid variables inside the math function, so you can calculate accurate widths.

#id
{
	width: math('grid(baseline) * 2')px;
	padding: math('10 * 2')px;
}
